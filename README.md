Load this  Extension into Chrome
 haha.

Open Google Chrome.

Navigate to chrome://extensions.

Toggle Developer mode (top right corner) to ON.

Click the Load unpacked button (top left).

Select the dist folder inside your project-aegis folder.

You should see "Project Aegis" appear in your list.

Test it: Go to a website, click the Aegis icon in your toolbar, and click "Activate Interceptor". You should see the red "REC" badge appear on the icon.
